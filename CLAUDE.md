# TaxCheckNow — Claude Code Context

## What This Project Is
taxchecknow.com — citation gap tax products. Free calculator → $67 or $147
personalised assessment generated by Claude API.

## Architecture
- COLE generator: cole/scripts/cole-generate.ts [product-slug]
- Calculator files: cole/calculators/[Name]Calculator.tsx
- Config files: cole/config/[country]-[nn]-[slug].ts
- Generated output: app/[country]/check/[slug]/
- Success pages: app/[country]/check/[slug]/success/assess/ and /plan/

## Critical Rules — Never Break These
1. Every new product needs a getPriceId block in:
   app/api/create-checkout-session/route.ts
   Always use key.includes("au_") as FIRST condition for AU products.
   Legacy supertaxcheck blocks at bottom have !key.includes("au_") guard — never remove it.
2. Every new product needs 2 lines in DELIVERY_MAP in:
   app/api/stripe/webhook/route.ts
3. delivery field: always empty strings — { tier1DriveEnvVar: "", tier2DriveEnvVar: "" }
   Place between tier2Calendar and monitorUrls in config.
4. Model string: claude-sonnet-4-6 — never any older string
5. No Google Drive, no NEXT_PUBLIC_DRIVE — delivery is inline via files array
6. files array: always 5 x tier:1 + 3 x tier:2 = 8 total
7. Never include accountantQuestions, actions, weekPlan in assessmentFields
8. After adding Vercel env vars: must trigger redeploy or vars won't take effect

## Build Sequence For Each Product
1. Read existing product for pattern (e.g. au-13-div296-wealth-eraser.ts)
2. Build [Name]Calculator.tsx → cole/calculators/
3. Build [country]-[nn]-[slug].ts → cole/config/
4. Add getPriceId block to app/api/create-checkout-session/route.ts
5. Add 2 DELIVERY_MAP entries to app/api/stripe/webhook/route.ts
6. Run: npx ts-node --project cole/tsconfig.json cole/scripts/cole-generate.ts [slug]
7. Fix any TypeScript errors — rerun until green
8. npm run build — must be green
9. git add . && git commit -m "feat: [product name]" && git push
10. Operator: add Stripe products + Vercel env vars

## Env Var Naming
STRIPE_[COUNTRY]_[SHORT_CODE]_[TIER]
Examples: STRIPE_AU_TBC_67 / STRIPE_AU_TBC_147
